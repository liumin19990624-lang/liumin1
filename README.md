# ANIME ENGINE v3.8 🎬

动画剧本创作引擎 - 一站式故事大纲、剧本生成、分镜设计工具，支持本地存储与 Word 导出。

## 核心功能

✅ 本地文件管理（小说原著、剧本、大纲）  
✅ 智能剧本生成（基于 DMXAPI 驱动）  
✅ 故事大纲自动提取  
✅ 工业级分镜表生成  
✅ 角色设定与小传创作  
✅ 剧本 Word 文档导出  
✅ 本地存储持久化（自动保存文件）

## 技术栈

- React + TypeScript
- Docx.js（Word 导出）
- DMXAPI（AI 生成能力）
- Tailwind CSS（UI 样式）

## 快速开始

### 前置要求
- Node.js 16+（推荐 18+）
- 网络连接（用于调用 DMXAPI）
